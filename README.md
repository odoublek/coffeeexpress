# Coffee Express Website

## Kurulum Talimatları

### 1. Node.js İndir
👉 https://nodejs.org adresine git → "LTS" versiyonu indir ve kur.

### 2. Projeyi Çalıştır
Terminali aç (VS Code'da Terminal > New Terminal):

```
npm install
npm start
```

Tarayıcı otomatik açılır: http://localhost:3000


### Klasör Yapısı
